Liquid Reverie

This website represents a culmination of my web development skills alongside my more creative aspirations. It is coded in a fairly simple framework (HTML/CSS/JavaScript) but the design is relatively complex and multi-sectional. Using the design principles learned from my human-computer interaction course in university, I've made sure to create something that is both visually engaging and functional. This web project acts simultaneously as a software development project and a digital portfolio for myself and my creative works.

Mass Spectrometry Hub

The aim of this project is the publication of the AdductHunter tool allowing users to use the tool and share their results. The website will enable users to take their Mass Spec data collected through research, run it through AdductHunter, and then publicize the results to the public and to groups that match their research niche.

Systematic Literature Review - Augmented Reality in Physics Education

This report presents and provides an analysis of six impactful papers in the field of augmented reality implementations for physics education. The applications detailed in these papers are comprehensive and each implementation provides the possibility to enhance the experience of students in physics education; unfortunately, there is a succinct lack of proof regarding the efficacy of these implementations in comparison to other virtual teaching methods as well as traditional teaching methods.
